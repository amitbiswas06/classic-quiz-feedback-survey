== Description ==

"Classic quiz feedback survey (CQFS)" is a simple WordPress plugin for quiz, feedback and survey. It is easy to use and maintain. You can create 3 types of forms with multiple choice questions. Questions can be assigned by categories. This plugin creates basically 3 custom post types.

1. Questions
2. Build
3. Entries

Where "questions" post type contains all questions that you create. "Build" post type creates the form for quiz, feedback or survey. The "entry" post type contains data submitted by user.

This plugin has no third party dependency. Plain JavaScript is used for form validation to ajax form submission. jQuery is not used at all.

**Show them everywhere.** Load and display your CQFS forms in any page or post with single shortcode!

== Installation ==

= Minimum Requirements =

* WordPress 5.1 or greater.
* PHP 7.1 or greater.

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Classic quiz feedback survey" Or Upload the entire `classic-quiz-feedback-survey` folder to the `/wp-content/plugins/` directory.
3. **Activate** Classic Quiz Feedback Survey from your Plugins page
4. **Click** on the new menu item "CQFS" and create your first Question and Create New Build then!
5. **Read** the documentation to [get started](https://templateartist.com/cqfs/)


== Frequently Asked Questions ==

= Where can I get support or talk to other users? =

If you get stuck, you can ask for help in the [CQFS Plugin Forum](https://wordpress.org/support/plugin/classic-quiz-feedback-survey).

= Will CQFS work with my theme? =

Yes! CQFS will work with any theme. Basic styling is also provided. However you can use your own styling to match your website or brand.

= Can I customize default texts or add my own HTML? =

Yes! Please find all about the [CQFS Hooks](https://templateartist.com/cqfs/action-hooks/) and [CQFS Filters](https://templateartist.com/cqfs/filter-hooks/)

= Can CQFS be translated in other languages? =

Yes! CQFS is translation ready. Please find the `POT` file inside the `languages` folder to start with.

= Where can I request new features and extensions? =

You can use [CQFS Plugin Forum](https://wordpress.org/support/plugin/classic-quiz-feedback-survey) to request new features.

= Where can I report bugs or contribute to the project? =

Report bugs on the [CQFS GitHub repository](https://github.com/amitbiswas06/classic-quiz-feedback-survey/issues). You can also report them in our [support forum](https://wordpress.org/support/plugin/classic-quiz-feedback-survey).
